## 📜 Fitur Utama
1.  **Sertifikat Interaktif:** Halaman sertifikat yang mendukung tampilan multi-halaman dengan navigasi titik (*dots*) dan panah.
2.  **Timeline Responsif:** Visualisasi perjalanan hidup yang interaktif menggunakan CSS transisi.
3.  **Desain Kreatif:** Header dengan efek *wave*, skema warna pastel yang lembut, dan desain *glassmorphism* pada navigasi.
4.  **Loader Animasi:** Transisi antar halaman yang halus dengan indikator loading kustom.
